# 1. Core Philosophy

All code generated by the AI agent must follow these fundamental principles:

## Modularity

- Break functionality into small, independent modules.
- Each module must have a single responsibility.
- Modules should be reusable across the application.

## Consistency

- Similar features must follow the same structure, naming pattern, and styling.
- UI components must follow the same layout, spacing, colors, and typography.

## Reusability

- If functionality or UI appears more than once, it must be abstracted into a reusable component.

## Maintainability

- Code should be readable, predictable, and easy for humans to modify later.

## Scalability

- Avoid shortcuts that make future expansion difficult.
